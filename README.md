# MKWD - Make Web Development Project

Professional web project scaffolding tool - Create FastAPI projects instantly!

## Installation
```bash
pip install git+https://github.com/DebugJedi/scaffolding_tool.git
```

## Usage
```bash
mkwd my_project              # Create portfolio
mkwd my_api --type=api       # Create API
mkwd my_app --type=fullstack # Create full-stack
```

## What It Creates

- Complete FastAPI backend
- Database integration (SQLAlchemy)
- Analytics tracking
- Security middleware
- API documentation
- Docker support
- 50+ files ready to use!
